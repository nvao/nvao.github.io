Primero debemos descargar y obtener todos los instaladores necesarios.


Primer paso: Ir a la pagina de Windows
Segundo paso: Conectar la unidad flash, usb o dvd
Tercer paso: Descargar y ejecutar la herramienta
Cuarto paso: Aceptar los terminos de licencia y avisos aplicables

Quinto paso: Elegir la opcion "crear medios de instalacion"
Sexto paso: Seleccionar el idioma, arquitectura y la edicion de Windows 10

Septimo paso: Elegir la unidad flash,USB,elegir archivo ISO

Octavo paso: Esperar la descarga y creacion del dispositivo de instalacion

Despues preparamos la PC para configurar la BIOS


Primer paso: Fijarse al entrar o encender la computadora, que tecla me deja ingresar a la BIOS

Segundo paso: Buscar las opciones de But o de Buteo
Tercer paso: Poner la unidad USD como primera opcion de Buteo

Cuarto paso: Salir de la BIOS guardando los cambios realizados

Por último hacemos la instalación del Sistema Operativo Windows 10


Uno: Presionar cualquier tecla para iniciar la instalacion,cuando aparezca el mensaje
Dos: Elegir el idioma,formato de hora y moneda y el metodo de entrada de teclado
Tres: Elegir la opcion de instalar ahora
Cuatro:rPoner la clave de activacion de windows, si no contamos con una, elegir la opcion no tengo clave de producto
Cinco: Elegir la version de Windows 10, Windows 10 pro
Seis: Aceptar los terminos de licencia de Windows
Siete: Elegir la opcion ""instalar solo windows
Ocho: Elegir la unidad para instalar windows,elegir la opcion formatear y luego siguiente
Nueve: Esperar que realice el proceso de instalacion
